# Приложение A. \(\mathbb{Z}_3\)-градуированная алгебра и троичная скобка Намбу

## A1.1. Алгебра наблюдаемых

Алгебра наблюдаемых \(\mathcal{A}\) разлагается в прямую сумму трёх подпространств:
\[
\mathcal{A} = \mathcal{A}_0 \oplus \mathcal{A}_1 \oplus \mathcal{A}_2,
\]
с градуировкой по модулю 3: \(\mathcal{A}_i \cdot \mathcal{A}_j \subseteq \mathcal{A}_{i+j \pmod{3}}\).

Инволюция \(*\) отображает \(\mathcal{A}_1 \leftrightarrow \mathcal{A}_2\) и оставляет \(\mathcal{A}_0\) инвариантным.

Физическая интерпретация:
- \(\mathcal{A}_1\) — «созидание» (\(+1\)),
- \(\mathcal{A}_2\) — «разрушение» (\(-1\)),
- \(\mathcal{A}_0\) — «потенциал» (\(0\)).

## A1.2. Оператор Меры

Оператор Меры \(\hat{M} \in \mathcal{A}_0\) имеет спектр \(\{+1,0,-1\}\) и представляется как
\[
\hat{M} = (+1)P_+ + (0)P_0 + (-1)P_-,
\]
где \(P_+,P_0,P_-\) — ортогональные проекторы.  
След \(\operatorname{Tr}\hat{M}\) (в факторе фон Неймана типа \(II_1\)) — глобальная топологическая память.

## A1.3. Троичная скобка Намбу

Для любых трёх элементов \(a,b,c \in \mathcal{A}\) троичная скобка определяется как
\[
[a,b,c] = abc + bca + cab - acb - cba - bac.
\]

Свойства:
- **Полная антисимметрия**:
  \[
  [a,b,c] = -[a,c,b] = -[b,a,c] = -[b,c,a] = -[c,a,b] = -[c,b,a].
  \]
- **Тождество Якоби для тройных скобок**:
  \[
  [a,b,[c,d,e]] = [[a,b,c],d,e] + [c,[a,b,d],e] + [c,d,[a,b,e]].
  \]

## A1.4. Редукция к коммутатору

Если один из аргументов пропорционален единичному оператору, скобка сводится к обычному коммутатору:
\[
[a,\mathbf{1},b] = 2(ab - ba) = 2[a,b]_{\text{comm}}.
\]
В высокодобротном пределе \(\hat{M} \approx m\mathbf{1}\), и уравнение динамики принимает вид
\[
\frac{dF}{d\tau} = m\,[F,H]_{\text{comm}}.
\]

## A1.5. Некоммутативные дифференцирования

Вместо координат вводятся дифференцирования \(D_0, D_+, D_-\) с коммутационными соотношениями, аналогичными квантовой группе \(U_q(\mathfrak{sl}(2))\) при \(q^3=1\):
\[
[D_0, D_+] = \alpha D_+,\quad [D_0, D_-] = -\alpha D_-,\quad [D_+, D_-] = \beta D_0 + \gamma \hat{M}.
\]
При усреднении по ансамблю узлов и в пределе \(\operatorname{Tr}\hat{M}\to\infty\) эти дифференцирования переходят в частные производные по координатам.

# Приложение A2. Вывод уравнения Шрёдингера и непрерывный предел

## A2.1. Высокодобротный предел

Добротность узла \(Q = \frac{E}{\Delta E}\) велика, когда диссипация мала. В этом пределе оператор Меры \(\hat{M}\) почти не меняется за период и может быть заменён на среднее значение \(\langle \hat{M} \rangle = m\mathbf{1}\). Тогда троичная скобка редуцируется:
\[
[F,\hat{M},H] \approx m\,[F,H]_{\text{comm}}.
\]

## A2.2. От дискретного к непрерывному времени

Переход к непрерывному времени \(t\) осуществляется заменой
\[
\frac{dF}{d\tau} = m\,[F,H] \quad\Rightarrow\quad i\hbar_{\text{eff}}\frac{dF}{dt} = [F,H],
\]
где \(\hbar_{\text{eff}} = 1/(m)\) — эффективная постоянная Планка, связанная с масштабом фазового градиента:
\[
\hbar_{\text{eff}} \sim \frac{1}{|\nabla\Phi^\phi|}.
\]

## A2.3. Картина Шрёдингера

В представлении, где \(\hat{M}\) диагонален (собственные состояния \(+1,0,-1\)), эволюция вектора состояния \(|\psi(t)\rangle\) определяется уравнением
\[
i\hbar_{\text{eff}}\frac{d}{dt}|\psi(t)\rangle = H|\psi(t)\rangle.
\]

Волновая функция \(\psi(\mathbf{r},t) = \langle \mathbf{r}|\psi(t)\rangle\) есть амплитуда вероятности нахождения фазового узла в точке \(\mathbf{r}\).

## A2.4. Координатное представление

Гамильтониан для электрона-хопфиона в непрерывном пределе имеет вид
\[
H = \int d^3x\left[ \frac{1}{2m_{\text{eff}}}(\nabla\Phi^\phi)^2 + V(\Phi^A) \right],
\]
где \(m_{\text{eff}}\) возникает из жёсткости поля (\(\Lambda_2,\Lambda_4\)), а \(V\) — из массового члена \(\Lambda_0(1-n_z)\).

В координатном представлении получается стандартное уравнение Шрёдингера:
\[
i\hbar_{\text{eff}}\frac{\partial\psi}{\partial t} = \left[ -\frac{\hbar_{\text{eff}}^2}{2m_{\text{eff}}}\nabla^2 + V(\mathbf{r}) \right]\psi.
\]

## A2.5. Связь с реальными константами

В модели \(\hbar_{\text{eff}} = \hbar\) и \(m_{\text{eff}} = m_e\) (масса электрона) возникают как геометрические параметры хопфиона при минимизации энергетического функционала (см. приложение B1).  
Таким образом, уравнение Шрёдингера — это **высокодобротное приближение** троичной динамики.
